Input/Output (io)¶

Introduction¶

ctapipe.io contains functions and classes related to reading data, like hessio_event_source. It currently contains some data structures related to the observatory layout, like CameraGeometry, but those will likely move soon to an instrument package.

Acces to Raw Data¶

The ctapipe.io.hessio package provides a convenient wrapper to reading simtelarray data files, like those used in CTA monte-carlo productions. It requires the pyhessio package to be installed (see Getting Started For Developers for instructions installing pyhessio).
