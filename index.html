CTA Experimental Pipeline Framework (ctapipe)¶

version: 0.0.dev602

What is ctapipe?¶

ctapipe is an experimental framework for the data processing pipelines for CTA.

Caution

This is not yet stable code, so expect large and rapid changes to structure and functionality as we explore various design choices.

  • Code, feature requests, bug reports, pull requests: https://github.com/cta-observatory/ctapipe
  • Docs: https://cta-observatory.github.io/ctapipe/
  • License: BSD-3
  • Python 3.4 or later (Python 2 is not supported)
